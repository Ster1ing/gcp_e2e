# gcp_e2e

## How to..
* install:
  * git
  * terraform
  * ansible
  * VSCode

* `gcloud auth application-default login`
* `cd terraform`
* `cp terraform.tfvars.dist terraform.tfvars`
* fill gaps in terraform.tfvars
* `terraform init`
* `terraform fmt`
* `terraform plan`
* `terraform apply`
